# How to Delete Individual Invoices

## ✅ **FEATURE ADDED - DELETE SINGLE INVOICE**

### 🗑️ **How to Delete One Invoice:**

1. **Go to Admin Panel**
   - Login as Super Admin, Admin, or Store Manager
   - Navigate to: `Admin` → `Billing`

2. **Find the Invoice**
   - You'll see a table with all invoices
   - Each row shows: Invoice #, Customer, Amount, Status, etc.

3. **Delete the Invoice**
   - Look at the **"Actions" column** (last column)
   - Click the **red trash icon** 🗑️ for the invoice you want to delete
   - **Confirmation dialog** will appear

4. **Confirm Deletion**
   - Dialog shows: "Are you sure you want to delete invoice INV-XXX?"
   - Click **"Cancel"** to abort
   - Click **"Yes, Delete Invoice"** to confirm deletion

5. **Invoice Deleted**
   - Invoice is removed from the list
   - Success message appears: "Invoice INV-XXX deleted"
   - Change is visible to ALL admin users immediately

### 🛡️ **Safety Features:**
- ✅ **Confirmation Dialog** - Prevents accidental deletion
- ✅ **Warning Message** - Clear alert about permanent deletion
- ✅ **Cancel Option** - Can abort at any time
- ✅ **Role Restrictions** - Only Super Admin, Admin, Store Manager can delete

### 👥 **Who Can Delete Individual Invoices:**
- ✅ **Super Admin** - Full delete access
- ✅ **Admin** - Full delete access  
- ✅ **Store Manager** - Full delete access
- ❌ **Sales Staff** - Cannot delete invoices
- ❌ **Inventory Manager** - Cannot delete invoices

### 📍 **Where to Find Delete Button:**
```
Invoice Table → Actions Column → Red Trash Icon 🗑️
```

### ⚠️ **Important Notes:**
- **Individual Delete**: Removes only one invoice
- **Global Clear**: "Clear Data" button removes ALL invoices
- **Cannot Undo**: Deleted invoices cannot be recovered
- **Audit Log**: All deletions are logged for tracking
